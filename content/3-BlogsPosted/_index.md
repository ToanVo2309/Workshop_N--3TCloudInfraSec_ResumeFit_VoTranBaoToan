The following are the blogs containing technical knowledge that I have posted to the AWS Study Group:

###  [Blog 1 - SYSTEM MONITORING ON AWS: STREAMING CLOUDWATCH METRICS TO OPENTELEMETRY COLLECTOR IN VPC USING LAMBDA](3.1-Blog1/)
This blog introduces a push-based observability solution using AWS Lambda, Amazon Data Firehose, and Network Load Balancer to stream metrics from AWS CloudWatch directly to an OpenTelemetry Collector inside a private VPC, helping optimize cost and eliminate vendor lock-in.

###  [Blog 2 - Hands-on Experience: Building an AI Gateway to Amazon Bedrock with Amazon API Gateway](3.2-Blog2/)
This blog shares practical experience designing a Serverless AI Gateway using Amazon API Gateway and AWS Lambda to manage access control, rate limiting, and cost tracking for Amazon Bedrock large language models (LLMs).

###  [Blog 3 - A Very Cool Feature of AWS Lake Formation](3.3-Blog3/)
This blog introduces a new AWS Lake Formation feature that solves the common data access discrepancy between Athena and EMR Spark by allowing Lake Formation to issue temporary credentials for direct S3 data access.
