### Week 7 Objectives:
* Understand serverless computing principles with AWS Lambda.
* Expose serverless logic to the internet via Amazon API Gateway.
* Integrate serverless execution logs with CloudWatch for tracing.

### Tasks to be carried out this week:
| Day | Task | Start Date | Completion Date | Reference Material |
| --- | --- | ---------- | --------------- | ------------------ |
| Monday | Study AWS Lambda architecture, runtimes, compute resource sizing, and invocation types. | 01/06/2026 | 01/06/2026 | [Video](https://www.youtube.com/watch?v=eOBqlydpXR0) <br> [AWS Lambda Developer Guide](https://docs.aws.amazon.com/lambda/latest/dg/welcome.html) |
| Tuesday | Learn Amazon API Gateway deployment models: REST API and HTTP API endpoints. | 02/06/2026 | 02/06/2026 | [API Gateway Developer Guide](https://docs.aws.amazon.com/apigateway/latest/developerguide/welcome.html) |
| Wednesday | Write a Python-based AWS Lambda function to automate custom workflows. | 03/06/2026 | 03/06/2026 | [Lambda programming with Python](https://docs.aws.amazon.com/lambda/latest/dg/lambda-python.html) |
| Thursday | Create and configure API Gateway routes to trigger the Python Lambda function. | 04/06/2026 | 04/06/2026 | [Trigger Lambda via API Gateway](https://docs.aws.amazon.com/apigateway/latest/developerguide/getting-started-with-lambda-integration.html) |
| Friday | Set up IAM Execution Roles for Lambda and analyze execution logs in CloudWatch. | 05/06/2026 | 05/06/2026 | [Lambda IAM Execution Roles](https://docs.aws.amazon.com/lambda/latest/dg/lambda-intro-execution-role.html) |

### Week 7 Achievements:
* Built a fully serverless, pay-per-use web backend without provisioning host servers.
* Designed clean RESTful endpoint routes linking web inputs directly to microservice tasks.
* Configured secure, narrow-scoped IAM roles granting execution permissions to serverless hosts.
* Utilized CloudWatch logs to debug lambda logic and analyze performance run durations.
